This project is a Secret Code Generator Web Application built using Python and Django, which implements the Caesar Cipher encryption technique to encode and decode messages.
About the Project:
Convert normal text into a secret coded message
Decode encrypted messages back to original text

This project is designed as a learning project to explore:
Django framework
Backend logic implementation

Technologies Used:
Python 
Django 
HTML & CSS 

Features:
Encrypt message using Caesar Cipher
Decrypt message using same shift key
User input through web interface
Simple and clean UI
Backend logic handled using Django

How It Works:
1.User enters a message
2.User selects a shift value (key)
3.Application processes each character:
Shifts letters forward (encryption)
Shifts letters backward (decryption)
4.Displays the result instantly
